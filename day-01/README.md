# Day 01 — Java Basics

First day of Java. Setting up the environment and covering the absolute fundamentals.

---

## Topics Covered

- Java program structure (`public class`, `main` method)
- Printing output using `System.out.println()`
- Variables and data types (`int`, `double`, `char`, `boolean`, `String`)
- Basic user input using `Scanner`
- Arithmetic operators

---

## What I Learned

- Every Java program starts with a `public class` matching the filename
- The `main` method is the entry point: `public static void main(String[] args)`
- Java is **statically typed** — variable types must be declared explicitly
- `Scanner` from `java.util` is used to read user input
- Java requires a semicolon at the end of every statement
