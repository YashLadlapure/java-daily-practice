# Day 7 — While Loop

## Topics Covered
- Basic `while` loop syntax and structure
- Counters and simple repetition
- Sum of 1..n using a while loop
- Finding factors of a number
- Computing `a^b` using repeated multiplication

## Files

| File | Description |
|------|-------------|
| `CounterDemo.java` | Counts from 1 to 5 using while |
| `SumOneToN.java` | Computes sum of numbers from 1 to n |
| `FactorsWhile.java` | Prints all factors of a given integer |
| `PowerUsingWhile.java` | Computes a^b using a while loop |

## Key Concepts

```java
int i = 1;

while (i <= n) {
    // body
    i++; // always update to avoid infinite loop
}
```

## How to Run

```bash
javac SumOneToN.java
java SumOneToN
```
