# ☕ Day 01 — Java Basics

> First day of Java! Setting up the environment and learning the absolute fundamentals.

---

## 📚 Topics Covered

- Java program structure (`public class`, `main` method)
- How to print output using `System.out.println()`
- Variables and data types (`int`, `double`, `char`, `boolean`, `String`)
- Basic user input using `Scanner`
- Arithmetic operators

| File | What It Does |

| `VariablesDemo.java` | Declares and prints different data types |
| `InputOutput.java` | Takes user input and prints it back |

---

## 💡 What I Learned

- Every Java program starts with a `public class` matching the filename
- The `main` method is the entry point: `public static void main(String[] args)`
- Java is **statically typed** — you must declare the type of every variable
- `Scanner` from `java.util` is used for user input
- Unlike JavaScript, Java requires a semicolon after every statement
